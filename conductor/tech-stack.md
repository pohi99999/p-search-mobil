# Tech Stack

## Frontend (Mobil & Keresztplatform)
- **Framework:** React Native + Expo (A gyors cross-platform fejlesztésért, natív modulok kezeléséért)
- **Nyelv:** TypeScript (A típusbiztonságért)
- **UI Könyvtár:** React Native Paper (Material Design 3 támogatás)
- **Navigáció:** React Navigation (@react-navigation/native-stack)

## Backend & Adatbázis (BaaS)
- **Szolgáltató:** Supabase
- **Adatbázis:** PostgreSQL
- **Hitelesítés:** Supabase Auth
- **Biztonság:** Row Level Security (RLS) irányelvek

## Automatizáció & Scraping (Külső háttérszolgáltatás)
- **Workflow Engine:** n8n (A B2B kutatásokhoz és automatikus pályázat gyűjtéshez)
- **Szkriptek:** Node.js, SQLite (Lokális web-rescue szkriptekhez)
